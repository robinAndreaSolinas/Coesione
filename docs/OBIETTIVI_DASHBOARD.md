### Sintesi copertura dashboard

- **Coperti direttamente** dalla dashboard:
  - Traffico sito / articoli: `articles-unique-users`, `articles-pageviews`, `articles-published-count`, `articles-printed-count`, `articles-digital-count`.
  - Performance social: `social-engagement-rate`, `social-views`, `social-audience`, `social-shares`, `social-comments`, `social-reach`.
  - Performance video: `video-audience` (stream), `video-minutes-watched`, `video-completion-rate` (view-through medio).
  - Performance newsletter: open rate, click rate, iscritti totali/attivi, numero totale di invii.
  - Coinvolgimento tramite sondaggi: `surveys-count`, `surveys-total-responses`, `surveys-completion-rate`, `surveys-average-responses`.

- **Non coperti (solo progetto / qualitativi / amministrativi)**:
  - Indicatori di coordinamento, qualità, gestione finanziaria e rischi (WP1).
  - Numero di eventi, workshop, raccomandazioni, strumenti interattivi.
  - Satisfaction / awareness / perception (studenti, docenti, stakeholder, comunità locali).
  - Numero di post social, eventi high-profile, policy briefs, ecc.

Questa tabella può essere estesa in futuro aggiungendo nuove colonne (“Come implementarlo in dashboard”, “Priorità”) per pianificare eventuali evoluzioni del prodotto.
